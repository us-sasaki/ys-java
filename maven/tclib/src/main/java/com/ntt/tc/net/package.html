このパッケージは、Things Cloud REST API を利用するのに便利な Java API を提供しています。Things Cloud 内のデータは、com.ntt.tc.data 以下のパッケージにあるクラスによって表されます。
  • API...createMeasurement, updateManagedObject, createLocationUpdateEvent のような API を Java メソッドとして提供します。
  • APIUtil...複数の REST API を組み合わせた、外部IDを指定して ManagedObject があれば作り、なければ生成するような便利メソッドを提供します。
  • Rest...RESTアクセスを提供します。
Things Cloud への REST API へのアクセスに外部パッケージを用いておらず、java 環境によらない軽量な利用が可能です。
