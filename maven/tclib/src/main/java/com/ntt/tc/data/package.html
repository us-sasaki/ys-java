このパッケージは、Things Cloud REST API にアクセスするのに便利な基本的なデータ構造を定義するクラスを提供します。REST API で利用される各種データを表すオブジェクトはサブパッケージで定義されます。本パッケージには、以下が含まれます。
  • C8yData...REST APIで利用される各種データの基底クラスとなり、JSON 形式との相互変換を行います。
  • TC_Date...REST APIで利用されるデータの time フィールドの形式(ISO-8601)と java.util.Date の相互変換を行うデータメンバです。
  • TC_Boolean, TC_Int...REST APIで利用される null 値を取りうる boolean, int を扱うデータメンバです。
