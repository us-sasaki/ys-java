// １．準備
// 　　○ＮＴとスーツコントラクトで分ける。
// 　　○スーツコントラクトのとき，切り札をトランプ、
// 　　　　切り札以外のスートをサイドスートと呼ぶ
// 　　○ダミーと自分について各スートのオリジナル枚数を覚えておく
// 　　○各スート、各プレイヤーが今までフォローした枚数を覚えておく
// 
// 　　○ディクレアラーとパートナーの各スートの枚数を調べる。
// 　　　（詳細）見える手：自分（Ｍ）とダミー（Ｖ）
// 　　　　　　　見えない手：ディクレアラー（Ｄ）とパートナー（Ｐ）
// 　　　　　　　あるスートのオリジナル枚数をＳｍ，Ｓｖ，Ｓｄ，Ｓｐと書くと
// 　　　　　　　Ｓｍ，Ｓｖ：既知
// 　　　　　　　Ｓｄ＋Ｓｐ＝１３−Ｓｍ−Ｓｖ
// 　　　　　　　なので、ＳｄまたはＳｐのどちらかが判明すれば
// 　　　　　　（どちらかがショウアウト＝スートにフォローしない、したとき判明）
// 　
// 　　　　　　　もう一方が分かる。このときＳｄ，Ｓｐを確定する
// 　　　　　　　
// 　　　　　　　３つのスートに関してＳｄ，Ｓｐが確定したら
// 　　　　　　　４スートについての和　ΣＳｄ＝１３、ΣＳｐ＝１３
// 　　　　　　　なので４つ目のスートについてもＳｄ，Ｓｐは判明する。
// 　　
// 　　○各スートのオリジナル枚数が判明したら、フォローした枚数から
// 　　　現在の枚数を調べておく
// 
// 　　○自分の手にあるウィナーカードを毎回考える（既存）
// 
// 　　○スーツコントラクトのときは、ダミー（またはディクレアラー）
// 　　　でラフされる分をウィナーと認めない
// 　　　　（詳細）
// 　　　　　・デフォルト
// 　　　　　ダミーにトランプが残っているとき、サイドスートの各スート
// 　　　　　についてダミー現在残っている枚数までのウィナーしかみとめない。
// 　　　　　ダミーに２枚のとき３枚以上のウィナーがあれば、上から２枚をウィナー
// 　　　　　として残し，３枚目はウィナーではないとする
// 
// 　　　　　・ダミーの方がディクレアラーより現在トランプが長いことが判明したら
// 　　　　　この操作はディクレアラーにラフされる分をウィナーと認めない、
// 　　　　　とする。修正はスートの枚数が確定しているものだけでよい。
// 
// 　　○ウィナーの数を数える。
// 　　　スーツコントラクトのときダミーとディクレアラーにラフされる分
// 　　　を数えないようにする。（ＮＴのときはただ数えるだけでいい）
// 　　　　　（詳細）
// 　　　　　・ダミーにラフされる分については上の修正で考慮済み。
// 　　　　　　ディクレアラーについては、
// 　　　　　　ディクレアラーのトランプの枚数＝＞１のとき
// 　　　　　（判明してないときは＝＞１が成り立っているものとする）、
// 　　　　　　ディクレアラーのサイドスートの各スートで枚数が判明している
// 　　　　　　ものにつきその枚数分までしかウィナーの数をカウントしない、
// 　　　　　　また、サイドスートのウィナーの合計数は
// 　　　　　　１３−ディクレアラーのトランプの枚数（判明していれば）
// 　　　　　　以下とする。　　　　　　　　　
// 
// 　　　　　・ダミーの方がディクレアラーよりトランプが長いと判明している
// 　　　　　　状況ではディクレアラーにラフされる分は上の修正で考慮済みなので
// 　　　　　　ダミーでラフされる分についてさらに考慮する
// 
// 　　○パートナーのウィナーを調べる
// 　　　パートナーがＫ、Ｑ、Ｊ，Ｔをリードしたときに考える。
// 　　　ただし予想したカードがすでにプレイされていたり、
// 　　　今後プレイされたらウィナーではなくなる。
// 
// 　　　・Ｋリード…そのスートのＡ（デフォルト）を持っているとして、
// 　　　　　　　　　Ａが見えたらＱを持っているとして、ウィナーかどうかを調べる
// 　　　　　　　　
// 　　　・Ｑリード…そのスートのＪを持っているとしてウィナーかどうかを調べる
// 　　　・Ｊリード…Ｔ保証で同様
// 　　　・Ｔリード…９保証で同様
// 
// 　　　ウィナーであれば覚えておく。
// 　　　ただし、自分のウィナーと同様ダミー（またはディクレアラーの場合もある）
// 　　　にラフされる分は認めないようにする。
// 
// 　　○パートナーのウィナーの数を数える。
// 　　　自分のウィナーの数に準じてラフされない分を数える。
// 　　　
// ２．Ｏ．Ｌ．（オープニングリード）
// ・ＮＴのとき、
// 次の優先順位でスートを決める
// １）指定がある場合
// ２）一番長いスート
// 
// スートが決まったらリードするカードを次のように選ぶ（ＮＴのオープニングリー
// ド）
// （カードまで指定されている場合を除く）
// 　・ＡＫＱ，ＫＱＪまたはＫＱＴからＫ
// 　・ＡＱＪＴ，ＡＱＪ９，ＱＪＴまたはＱＪ９からＱ
// 　・ＡＫＪＴからＡ
// 　・ＫＪＴ，ＪＴからＪ
// 　・ＡＫＴ９、ＡＴ９，ＫＴ９，ＱＴ９、ＡＱＴ９，Ｔ９からＴ
// 　・その他：４ｔｈベスト（上から４枚目）
// 
// 
// ・スーツコントラクトのとき
// 次の優先順位
// ０）指定がある場合はそれ
// １）トランプスートを除外
// ２）ＡＫのあるスートがあればそのＫ
// ３）ＫＱのあるスートがあればそのＫ
// ４）シングルトン（Ａ、Ｋ、Ｑはだめ）
// ５）ＱＪのあるスートがあればそのＱ（ＡＱＪならだめ）
// ６）ダブルトン（Ａ，Ｋ，Ｑがあったらだめ）は上
// ７）何でもいい
// 
// 「何でもいい」でスートが決まった場合のときのＯ．Ｌ．の決め方
// 　（１）ＡがあればＡ
// 　（２）ＫＪＴ、ＪＴからはＪ
// 　（３）ＫＴ９，ＱＴ９、Ｔ９からはＴ
// 　（４）９以下が頭ならトップ
// 　（５）それ以外：４枚以上なら４枚目、３枚：３枚目、２枚：上
// 
// オープニングリードのカードは覚えておく。
// 
// ３．ディフェンダーが勝ったときのリード
// ・ＮＴの場合
// 　スートの決め方次の順位：
// 　（０）自分の手のウィナーの数＋パートナーのウィナーの数が
// 　　　　コントラクトを落とすのに十分なとき、自分のウィナーをキャッシュ
// 　（１）パートナーにウィナーのあるスート
// 　（２）Ｏ．Ｌ．と同じスート
// 　（３）今までディフェンダーが勝ったときにリードしたスート（最近から順に）
// 　（４）０〜３のスートがないとき
// 　　　　ＬＨＯ（ディクレアラーの左手）：
//                ダミーのアナー（ＨＣＰで判断）の多いスート
// 　　　　ＲＨＯ（ディクレアラーの右手）：
// 　　　　　　　　ダミーのアナー（ＨＣＰで判断）の少ないスート　　
// 
// 　スート内でのカードの決め方を次の順位で決める　　　　　　　　
// 　　（１）そのスートの中で、トップがウィナーならキャッシュ
//　　　　　ＡＫからはＫ、他は上
// 　　（２）ＫＱからＫ
// 　　（３）ＱＪからＱ
// 　　（４）ＫＪＴ，ＪＴからＪ
// 　　（５）ＫＴ９，ＱＴ９、Ｔ９からＴ
// 　　（６）その他：現在２枚：上
// 　　　　　　　　　現在３枚：３枚目
// 　　　　　　　　　現在４枚以上→４枚目　
// 
// ・スーツの場合
// 　スートの決め方次の順位：
// 　（０）自分の手のウィナーの数＋パートナーのウィナーの数が
// 　　　　コントラクトを落とすのに十分なとき、すべてキャッシュ
// 
// 　（１）ラフリスのスートを除外（それしかなければしかたない）
// 　　　　ラフリスのスートとは：
// 　　　　ダミーにもディクレアラーにもトランプが残っている状況で
// 　　　　　　　　（現在０枚と判明していないこと）　　　　
// 　　　　ダミーもディクレアラーもが現在０枚と判明しているスート　　　　　　　
// 　　　　　　　　　
// 　（２）パートナーのトランプスートが現在０枚と確定しないとき、かつ
// 　　　　パートナーに現在０枚と確定しているサイドスートがあるとき、そのスート
// 　　　　（ラフさせる）
// 
// 　（３）パートナーにウィナーのあるスート
// 　（４）Ｏ．Ｌ．と同じスート
// 　（５）いままでディフェンダーが勝ったときにリードしたスート（最近から順に）
// 　（６）以上のスートがないとき
// 　　　　ＬＨＯ（ディクレアラーの左手）：
// 　　　　　　　ダミーのアナー（ＨＣＰで判断）の多いスート
// 　　　　ＲＨＯ（ディクレアラーの右手）：　　　　　　　
// 　　　　　　　ダミーのアナー（ＨＣＰで判断）の少ないスート　　　　　　
// 
// 　スート内でのカードの決め方
// 　次の順位で決める　　　　　　　　
// 　　（１）そのスートの中で、トップがウィナーならキャッシュ
// 　　　　　ＡＫからはＫ、他は上
// 　　（２）ＫＱからＫ
// 　　（３）ＱＪからＱ
// 　　（４）ＫＪＴ，ＪＴからＪ
// 　　（５）ＫＴ９，ＱＴ９、Ｔ９からＴ
// 　　（６）その他：現在２枚：上
// 　　　　　　　　　現在３枚：３枚目
// 　　　　　　　　　現在４枚以上→４枚目
// 
// ４．フォローのしかたは今まで通りでいい
//
